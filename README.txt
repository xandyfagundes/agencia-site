===========================================
ALFA EMPREENDIMENTOS DIGITAIS - HTML ESTÁTICO
===========================================

Este pacote contém os arquivos HTML estáticos do site da Alfa Empreendimentos Digitais,
prontos para serem hospedados em qualquer servidor web.

ARQUIVOS INCLUÍDOS:
-------------------
- index.html       → Página principal do site
- styles.css       → Folha de estilos CSS
- script.js        → JavaScript para animações e interações
- README.txt       → Este arquivo (instruções)
- alfa-logo.png    → Logo da empresa (copie da pasta src/assets)
- hero-bg.jpg      → Imagem de fundo do hero (copie da pasta src/assets)

ANTES DE FAZER UPLOAD:
----------------------
1. Copie os arquivos de imagem:
   - Copie "src/assets/alfa-logo.png" para a pasta "static-html/"
   - Copie "src/assets/hero-bg.jpg" para a pasta "static-html/"

2. Certifique-se de que todos os arquivos estão na mesma pasta:
   static-html/
   ├── index.html
   ├── styles.css
   ├── script.js
   ├── alfa-logo.png
   ├── hero-bg.jpg
   └── README.txt

COMO HOSPEDAR:
--------------

OPÇÃO 1 - Hospedagem Compartilhada (cPanel, Hostinger, etc):
1. Faça login no painel de controle da sua hospedagem
2. Acesse o gerenciador de arquivos
3. Navegue até a pasta "public_html" ou "www"
4. Faça upload de todos os arquivos (index.html, styles.css, script.js, e imagens)
5. Acesse seu domínio para ver o site funcionando

OPÇÃO 2 - GitHub Pages:
1. Crie um repositório no GitHub
2. Faça upload de todos os arquivos
3. Vá em Settings > Pages
4. Selecione a branch main e a pasta root
5. Seu site estará disponível em: usuario.github.io/nome-repositorio

OPÇÃO 3 - Netlify/Vercel:
1. Crie uma conta no Netlify (netlify.com) ou Vercel (vercel.com)
2. Arraste a pasta "static-html" para o upload
3. Pronto! Seu site estará no ar em segundos

OPÇÃO 4 - FTP:
1. Use um cliente FTP como FileZilla
2. Conecte-se ao seu servidor usando as credenciais FTP
3. Faça upload de todos os arquivos para a pasta raiz do site
4. Acesse seu domínio

PERSONALIZAÇÕES:
----------------
Para alterar textos, cores ou conteúdo:
1. Abra o arquivo index.html em um editor de texto
2. Modifique os textos conforme necessário
3. Para alterar cores, edite o arquivo styles.css na seção ":root"
